# ✈ 飞机大战

浏览器端射击小游戏，可直接在线游玩。

## 在线游玩

**[点击开始游戏 →](https://sw937.github.io/plane-battle/)**

## 操作说明

| 操作 | 键盘 | 触屏 |
|------|------|------|
| 移动 | ← → 或 A D | 屏幕左右按钮 |
| 射击 | 空格 | 射击按钮 |
| 暂停 | P | — |

## 本地运行

```bash
./start.sh
```

或双击 `启动游戏.command`，浏览器访问 `http://127.0.0.1:8912`。

## 技术栈

纯 HTML5 Canvas + JavaScript（ES Module），无需构建工具。
